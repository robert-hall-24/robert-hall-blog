Control Flow and Loops are used in code to make it behavior in a specific order until you reach the result wanted. Lets say you want to walk to the shop. You know the route so you follow the directions you normally take. There are different mechanisms involved in control flow but lets use loops. When even is the DOM?

DOM meaning?


DOM stands for The Document Object Model (DOM).
The DOM is like a blueprint for that building, but instead of showing how pipes and wires are laid out, it shows how the different parts of the page fit together. This blueprint (DOM) lets programmers change things on the webpage, like moving furniture (content) around.


How is DOM used?


Lets think a of webpage. It is full of elements. What if you didn't need to change the HTML or CSS files. but still make change??
Can you do this?
Yes!! you can. The whole point of the DOM is for you to make changes to the content on your page by manipulating elements. Which is pretty is cool.
